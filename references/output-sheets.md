# 输出工作表

默认工作簿应至少包含：

- 年度纳税汇总
- 财产转让计税情景
- FIFO已实现盈亏
- 移动平均已实现盈亏
- 盈亏方法差异
- 年末剩余批次
- 持仓数量对账
- 期初逐月持仓对账
- 期初成本批次
- 股息与预扣税
- 融资利息应计
- 融资利息实际支付
- 全部交易
- 全部资金流水
- 全部持仓
- 月度覆盖
- 年末汇率
- 复核就绪性
- 文件追溯
- 版本信息

关键勾稽：

1. 年度汇总中的股息、抵免候选和自动抵免应与股息明细一致。
2. 自动抵免在无合格凭证时必须为 0。
3. FIFO、移动平均和三种亏损抵减情景应从逐笔处置明细独立汇总。
4. 未实现盈亏不得进入年度财产转让汇总。
5. 缺少年末汇率时，所有受影响的 CNY 字段应为空而不是 0。
6. 期初逐月持仓对账和年末持仓对账应分别核对。
