# 税务边界与强制表述

本工具生成中国内地税收居民、单一长桥账户的复核工作底稿，不构成法律或税务意见。

## 可以作为计算事实输出

- 月结单原始交易、费用、资金流水、持仓和公司行动。
- 从真实成交和费用重建的 FIFO 与移动加权平均成本账。
- 只包含已实现处置的盈亏；未实现盈亏不纳入。
- 股息税前金额、现金到账、月结单显示的单独或内嵌预扣税。
- 融资利息应计口径与实际支付口径。
- 经来源记录支持的年末汇率折算。

## 必须显示为“情景”或“待确认”

- FIFO 或移动加权平均哪一种作为最终申报方法。
- 跨交易、跨标的、跨市场的亏损抵减方式。
- 现金奖励的最终所得分类。
- 发行人、基金、ADR、期权、权证的所得来源地未有充分证据时。
- 融资利息是否可扣除；默认不扣除。
- 股息相关服务费是否可冲减收入；默认单独列示、不冲减。

## 税额抵免红线

- 月结单显示的预扣税只作为“抵免候选”。
- 没有合格完税证明、缴款书、纳税记录或主管机关接受的其他材料时，自动无条件抵免额必须为 0。
- 不得把“抵免候选后的税额”表述为最终应补税额。

## 缺失数据红线

- 缺少汇率：CNY 输出为空并阻断复核，不得写 0。
- 未知 PDF 版式：阻断结构化税务输出，不得套用最接近模板。
- 期初成本无法由历史成交重建：标记不完整或阻断，不得采用负数或不可解释的券商展示成本。
- 税务口径没有可归档依据：保留并列情景，不得标记“税法已确认”。
